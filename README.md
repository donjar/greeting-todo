# greeting-todo

A todo list in Fish's greeting message

## Install

With [fisherman]

```
fisher donjar/greeting-todo
```

## Usage

Add todo named "clean house"
```fish
addtodo clean house
```

Delete the 2nd todo
```fish
deletetodo 2
```

Clear all todos
```fish
cleartodo
```

[fisherman]: https://github.com/fisherman/fisherman
